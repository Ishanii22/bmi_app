# BMI Predictor App 🚀

A simple Machine Learning web application that predicts Body Mass Index (BMI) based on user input (height and weight). This project demonstrates the integration of Machine Learning with a basic web interface.


## 📌 Features
- Input height and weight
- Predict BMI value instantly
- Simple and user-friendly web interface
- Machine Learning model trained using Python
- Lightweight and easy to run locally


## 🧠 How It Works
The application takes user input (height and weight), processes it through a trained ML model, and outputs the predicted BMI value.


## 🛠 Tech Stack
- Python 🐍
- Scikit-learn 🤖
- Pandas / NumPy
- HTML / CSS
- Flask / Streamlit (if used)



## 📊 Dataset
- The dataset consists of height and weight values used to calculate BMI.
- A labeled dataset was used to train the model for prediction.
